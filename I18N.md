# 🌍 Internacionalización (i18n) - RewardsHub

## ✅ Implementación Completada

Se ha implementado un sistema completo de internacionalización que:
- ✅ Detecta automáticamente el idioma del navegador
- ✅ Soporta Español (ES) e Inglés (EN)
- ✅ Permite cambio manual de idioma
- ✅ Persiste la selección en localStorage

## 🔧 Configuración

### **1. Archivo de Configuración (`src/i18n.js`)**

```javascript
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import LanguageDetector from 'i18next-browser-languagedetector';

i18n
  .use(LanguageDetector)  // Detecta idioma del navegador
  .use(initReactI18next)
  .init({
    resources: { es: {...}, en: {...} },
    fallbackLng: 'es',    // Idioma por defecto
    detection: {
      order: ['localStorage', 'navigator', 'htmlTag'],
      caches: ['localStorage'],
    }
  });
```

### **2. Inicialización en `main.jsx`**

```javascript
import './i18n' // Importar ANTES de App
```

## 🎯 Cómo Usar en Componentes

### **Método 1: Hook useTranslation**

```javascript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();

  return (
    <div>
      <h1>{t('common.welcome')}</h1>
      <p>{t('landing.description')}</p>
      
      {/* Con variables */}
      <p>{t('client.home.welcome', { name: 'Juan' })}</p>
      {/* Resultado: "¡Hola, Juan!" */}
    </div>
  );
}
```

### **Método 2: Trans Component (para HTML)**

```javascript
import { Trans } from 'react-i18next';

<Trans i18nKey="signup.client.areYouBusiness">
  ¿Eres un negocio? <Link to="/signup/business">Regístrate aquí</Link>
</Trans>
```

## 🔄 Cambiar Idioma Programáticamente

```javascript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { i18n } = useTranslation();

  const changeToEnglish = () => {
    i18n.changeLanguage('en');
  };

  const changeToSpanish = () => {
    i18n.changeLanguage('es');
  };

  // Obtener idioma actual
  const currentLang = i18n.language; // 'es' o 'en'

  return (
    <button onClick={changeToEnglish}>
      Switch to English
    </button>
  );
}
```

## 🌐 Detección Automática de Idioma

El sistema detecta el idioma en este orden:

1. **localStorage** - Si el usuario ya eligió un idioma
2. **navigator.language** - Idioma del navegador
3. **htmlTag** - Atributo lang del HTML
4. **fallback** - Español (por defecto)

### **Ejemplos de Detección:**

```javascript
// Navegador en inglés
navigator.language = 'en-US' → Idioma: EN

// Navegador en español
navigator.language = 'es-MX' → Idioma: ES

// Navegador en francés (no soportado)
navigator.language = 'fr-FR' → Idioma: ES (fallback)
```

## 🔄 Agregar Nuevos Idiomas

### **1. Crear archivo de traducción:**

```bash
src/locales/fr/translation.json
```

### **2. Agregar al i18n.js:**

```javascript
import translationFR from './locales/fr/translation.json';

const resources = {
  es: { translation: translationES },
  en: { translation: translationEN },
  fr: { translation: translationFR }  // Nuevo idioma
};
```

### **3. Actualizar LanguageSwitcher:**

```javascript
<button onClick={() => changeLanguage('fr')}>
  🇫🇷 FR
</button>
```

## 📋 Ejemplo Completo: Login con i18n

```javascript
import { useTranslation } from 'react-i18next';
import LanguageSwitcher from '../components/LanguageSwitcher';

function Login() {
  const { t } = useTranslation();

  return (
    <div>
      {/* Selector de idioma */}
      <LanguageSwitcher />

      {/* Título traducido */}
      <h1>{t('login.title')}</h1>
      <p>{t('login.subtitle')}</p>

      <form>
        {/* Labels traducidos */}
        <label>{t('common.email')}</label>
        <input type="email" placeholder={t('common.email')} />

        <label>{t('common.password')}</label>
        <input type="password" placeholder={t('common.password')} />

        {/* Botón traducido */}
        <button>{t('login.button')}</button>
      </form>

      {/* Link traducido */}
      <Link to="/forgot-password">
        {t('common.forgotPassword')}
      </Link>
    </div>
  );
}
```

## 🎯 Mejores Prácticas

### ✅ **DO:**
- Usar claves descriptivas: `client.home.welcome`
- Agrupar por sección: `common`, `client`, `business`
- Usar variables para nombres: `{t('welcome', { name })}`
- Mantener traducciones sincronizadas

### ❌ **DON'T:**
- Hardcodear textos: `<h1>Bienvenido</h1>`
- Usar claves genéricas: `text1`, `label2`
- Mezclar idiomas en el mismo archivo
- Olvidar actualizar ambos archivos (ES y EN)
